The .xml files are all the headers, .img are where the data are stored.
 
These runs were conducted with an atmosphere with Doose haze, just in the VIMS windows, with a Lambertian surface with 0.1 surface albedo. The filenames include “p***”: that number is the phase angle.
_colorCCD files are the ones you want for the images
_geo files are all the backplane information
_error and _SNR should probably just be ignored; pretty sure the “error” calculation wasn’t working correctly when this was run
